# 🎬 Movie App

> Status: 🚧 Projeto em andamento

Aplicação para listar e buscar filmes usando a API do TMDb. Desenvolvido com React, HTML, Tailwind CSS e Axios.

## Funcionalidades

- 🔎 Busca de filmes
- 🎞️ Visualização de detalhes
- ⭐ Favoritar

## Tecnologias

- React
- Tailwind CSS
- Axios
- TMDb API
- HTML
